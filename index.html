FPM - Foliocean Package Manager
Backend
Node segítségével fejlesztett NPM-hez hasonló, de ahhoz messze nem eléggé fejlett Privát Manager rendszer. Meghatározott Git Repo subrepository-ként történő importálását követően elérhetővé válnak az általunk írt kódok / modulok / komponensek függetlenül a fejlesztési nyelvtől. Különlegessége, hogy adott projektben gond nélkül módosíthatunk egy modulon anélkül, hogy egy modul frissítés felülírná az alkotásunkat.

Néhány elérhető funkció:
  • Hozzáadás - Legfrissebb vagy adott verzió
  • Törlés - Még üres mappákat sem hagyunk hátra
  • Frissités - Ha --conflict kapcsolóval kerül használatra, akkor git conflitot generálunk egy fájlban, ha ott eltérés található

Munkaszervezés és munkaidő nyílvántartás

Fullstack

Egy cég életében a hónap záráshoz tartozó adminisztrációk tarthatnak akár 1 hétig is. Kezdetben ezt a próblémát orvosoltam ezzel a rendszerrel és sikerült az emlitett folyamatot fél órára szűkítenem. A kollégák telefont használva bejelentkeznek és feltudják vinni a munkaidejüket, szabadságukat. Igy ha mindenki feltölti az adatait, akkor csak exportálni kell a rendszerből.

Később külön kérésre bővült a rendszer a munkaszervezési nyilvántartással, amely segitségével a kollégák már a projekt kiajánlásakor értesülnek a meghatározott feladatokról. Emellett egy harmadik rendszerrel történt API összeköttetésnek köszönhetően olyan dinamikus dokumentumokhoz is hozzáférnek digitálisan, amelyekkel eddig csak a nyomtatót terheltük.
