FPM - Foliocean Package Manager
Node segítségével fejlesztett NPM-hez hasonló, de ahhoz messze nem eléggé fejlett Privát Manager rendszer. Meghatározott Git Repo subrepository-ként történő importálását követően elérhetővé válnak az általunk írt kódok / modulok / komponensek függetlenül a fejlesztési nyelvtől. Különlegessége, hogy adott projektben gond nélkül módosíthatunk egy modulon anélkül, hogy egy modul frissítés felülírná az alkotásunkat. Ahány projekt, annyi probléma megoldása.
Munkaszervezés és munkaidő nyílvántartás
Egy cég életében a hónap záráshoz tartozó adminisztrációk tarthatnak akár 1 hétig is. Ezzel a rendszerrel sikerült ezt fél órára szűkíteni. Persze ehhez a kollégáknak támogatása is szükséges, de a csapatmunkánál nincs kifizetődőbb. Ez a Laravel rendszer könnyen kezelhető, bármilyen statisztikai adat kinyerhető belőle, használata kényelmes és bárhonnan elérhető. És azzal, hogy összeköthető RMS rendszerrel, a cég életében a papír felhasználás is radikálisan csökkent.
